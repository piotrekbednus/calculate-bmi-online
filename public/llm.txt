# Calculate BMI Online

This website provides a free, fast, and secure Body Mass Index (BMI) calculator.

## Features
- Calculate BMI using Metric units (kg, cm)
- Calculate BMI using Imperial units (lbs, ft/in)
- Instant results with BMI score and category (Underweight, Normal, Overweight, Obesity)
- No registration required
- Privacy-focused (all calculations are client-side)

## How to use
1. Select your preferred unit system (Metric or Imperial).
2. Enter your weight and height.
3. Click "Calculate BMI".
4. View your result and category.

## API
This is a static site and does not offer a public API. All logic is contained within the HTML/JS.

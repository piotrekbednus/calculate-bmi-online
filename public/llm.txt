# GeneratePassword.online

This website is a free online password generator. It allows users to create strong, secure passwords with customizable options such as length, uppercase/lowercase letters, numbers, and symbols.

## Features
- Customizable password length (4-64 characters)
- Option to include/exclude uppercase, lowercase, numbers, and symbols
- Option to exclude similar or ambiguous characters
- Password strength indicator
- Client-side generation (passwords are never sent to a server)
- Multi-language support

## API
There is no public API for this tool. All logic is performed client-side.
